# 🚀 Engenharia de Instrução: O Domínio dos Prompts
 ## 📝 Descrição do Projeto
Este projeto foi desenvolvido como uma atividade prática de **Engenharia de Instrução**, onde o objetivo central é dominar a técnica de comunicação com modelos de linguagem (LLMs) para obter resultados de alta precisão no menor tempo possível. A atividade desafiou os participantes a atuarem como engenheiros de prompt, utilizando o modelo Qwen para atingir "alvos" específicos (uma imagem artística e um e-mail formal complexo) dentro de um limite estrito de apenas 5 iterações.

A metodologia aplicada baseou-se no **Ciclo de Refinamento**, que consiste em analisar criticamente cada resposta da IA para integrar aprendizados — como a mudança de uma única palavra ou a adição de contexto — na instrução seguinte. O projeto demonstra como o ajuste fino de instruções pode transformar uma resposta genérica em um output altamente especializado.

 ## 🚀 Tecnologias Utilizadas
* **Modelo de IA:** Qwen (https://qwen.ai/)
* **Ferramentas:** Dispositivos móveis para interação em tempo real
* **Técnicas de Prompting:** Personas (Pirata com MBA), Contextualização Geográfica e Restrições de Formatação
 ## 📊 Resultados e Aprendizados
O projeto alcançou o sucesso ao converter instruções simples em comandos complexos que guiaram a IA para resultados específicos.
* **Controle de Composição Visual:** Foi identificado que a IA tendia a gerar o astronauta sentado; a integração de uma "cama branca" ao contexto permitiu alterar a postura do personagem para "deitado" conforme desejado.
* **Refinamento de Persona e Tom:** A evolução de um e-mail genérico para uma comunicação formal de "Capitão Horácio", incluindo anexos simulados e terminologia de gestão (stakeholders, ativos circulantes), demonstrou o poder de atribuir formações acadêmicas à persona (MBA em Letras).
* **Redução de Ruído:** O uso de instruções negativas (ex: "não queremos a imagem cortada na cintura") foi fundamental para atingir a enquadramento de corpo inteiro.

 ## 🔧 Como Executar
1. Acesse o modelo Qwen via celular.
2. Defina o "Alvo" (ex: Imagem do astronauta ou e-mail do pirata).
3. Execute o **Ciclo de Refinamento**:
    * **Avaliar:** Verifique se o tom está correto ou se detalhes foram ignorados.
    * **Integrar:** Conecte o erro ao novo aprendizado para construir a próxima instrução.
4. Respeite o limite de 5 prompts para alcançar o objetivo final.
