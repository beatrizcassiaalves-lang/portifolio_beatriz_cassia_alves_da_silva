
# 🎬 A corrida do Prompt
 
## 📝 Descrição do Projeto
Em trios, nós geramos um resultado específico (texto ou imagem, conforme definido pelo professor) utilizando o celular. O grande desafio? Vocês têm um limite de apenas 5 iterações (tentativas) para chegar à perfeição.
 
Desenvolvido como parte da disciplina de **Inteligência Artificial (2026)**, o sistema processa comandos do ser humano e processa eles como entender.
 
 
## 🚀 Tecnologias Utilizadas
* **Inteligência Artificial:** Gemini

 
## 🔧 Como Executar
1. Crie prompts refinados
2. Utilize a Inteligência Artificial de preferência.
3. Melhore sempre que ele gerar um resultado menos esperado.
